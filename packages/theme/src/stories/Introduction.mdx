# Orion UI Design System

Welcome to the Orion UI Design System documentation. This package (`@orion-ui/theme`) defines the core design tokens (colors, typography, shadows, spacing) used across the Orion UI component library.

## Installation

```bash
bun add @orion-ui/theme
```

## Usage

Import the CSS in your app:

```css
@import '@orion-ui/theme';
```

## Philosophy

The design system is built on [Tailwind CSS v4](https://tailwindcss.com/) and uses [Radix Colors](https://www.radix-ui.com/colors) for a consistent and accessible color palette.
